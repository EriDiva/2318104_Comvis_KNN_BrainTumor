# Klasifikasi Citra Tumor Otak Menggunakan Metode K-Nearest Neighbor (KNN)

## Metode
Menggunakan algoritma K-Nearest Neighbors (KNN) dengan preprocessing data, normalisasi (StandardScaler), pembagian data train-test, serta evaluasi menggunakan accuracy dan confusion matrix.
